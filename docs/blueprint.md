# **App Name**: CodeCraft Quest

## Core Features:

- Daily Coding Challenges: Automatically generate daily coding challenges with scaling difficulty.
- Code Library: Access a library of coding tasks categorized by domain and difficulty.
- Code Execution Engine: Execute code in Python, C++, Java, and JavaScript.
- XP and Level System: Earn XP for completing tasks and level up.
- Code Dungeon RPG: Embark on a coding adventure in a digital dungeon; each room presents a coding challenge to defeat monsters and progress.
- AI Code Assistant Tool: Leverage AI to get hints and explanations for solving coding problems. The AI may intelligently reason whether to use this to offer solutions based on the complexity.
- Profile and Leaderboard: Track progress, earn badges, and compete with classmates on a leaderboard.

## Style Guidelines:

- Primary color: HSL (53, 94%, 54%) which converts to RGB Hex: #FCE73B; evoking the feeling of enlightenment that comes with solving a tough problem.
- Background color: Desaturated yellow HSL (51, 20%, 93%) which converts to RGB Hex: #F5F3E1, providing a non-straining backdrop.
- Accent color: HSL (23, 94%, 54%) which converts to RGB Hex: #FCA03B; creating eye-catching accents and CTAs.
- Headline font: 'Space Grotesk', sans-serif. For a techy, modern feel, best suited for short texts like titles.
- Body font: 'Inter', sans-serif. Because it's neutral and works well for longer text blocks, should any occur.
- Code font: 'Source Code Pro' monospace; appropriate for showing the code to be worked on, in the challenge questions.
- Use pixel-art-inspired icons to enhance the gamified, dungeon-crawling experience.
- Implement smooth transitions and interactive animations for user feedback and engagement.
